# 🚆 Railway Ticket Reservation System

A modern Railway Ticket Reservation System developed using **C++**, **Object-Oriented Programming (OOP)**, **HTML**, **CSS**, and **JavaScript**.

This project started as a console-based C++ application and was later converted into a modern web-based application with an interactive UI and live deployment using Vercel.

---

# 🌐 Live Demo

🔗 [Live Website](https://railway-ticket-app-mu.vercel.app/)

# 📌 Project Overview

The Railway Reservation System allows users to:

✅ View available trains  
✅ Book train tickets  
✅ Check ticket details  
✅ Calculate ticket charges  
✅ Validate train numbers  
✅ Generate reservation confirmation  
✅ Display train information in tabular format  

The project demonstrates:
- Object-Oriented Programming
- Frontend Web Development
- UI Design
- JavaScript DOM Manipulation
- Deployment using Vercel
- GitHub Project Hosting

# 🏗️ Classes Implemented in C++

## 👤 Passenger Class
Stores:
- Passenger Name
- Train Number
- Number of Seats
- Ticket Charges

## 🚆 Train Class
Stores:
- Train Number
- Train Name
- Destination
- Price
- Departure Time
- Available Seats

## 🖥️ TrainReservationSystem Class
Handles:
- Login System
- Train Management
- Ticket Reservation
- Train Search
- Reservation Storage
- Menu Operations
